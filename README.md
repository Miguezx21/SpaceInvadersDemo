\# Space Invaders Demo - Unity 6



Este es un proyecto de práctica desarrollado en \*\*Unity 6 (6000.0.38f1)\*\*. Es un juego tipo Galaga/Space Invaders donde el jugador controla una nave para destruir oleadas de enemigos.



\## 🚀 Características actuales

\- \*\*Sistema de Salud:\*\* Controlado por el script `PlayerHealth.cs`, con una interfaz visual de corazones.

\- \*\*Game Manager:\*\* Maneja las condiciones de victoria (destruir todos los aliens) y derrota.

\- \*\*Gráficos:\*\* Estilo Pixel Art (naves, proyectiles y efectos de explosión).

\- \*\*Control de Versiones:\*\* Repositorio gestionado con Git y alojado en GitHub.



\## 🛠️ Tecnologías utilizadas

\- \*\*Motor:\*\* Unity 6

\- \*\*Lenguaje:\*\* C# tradicional

\- \*\*Repositorio:\*\* Git / GitHub
